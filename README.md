# Customer Support Data Analysis
## Tools Used
- Google Sheets
## Objective
To analyze customer support ticket data and identify key trends in customer issues and resolution performance.
## Key Analysis
- Tickets by Type
- Tickets by Status
- Ticket Priority Distribution
## Key Insights
- Most tickets are related to billing and technical issues
- 2769 tickets are closed, while 2819 are open and 2881 are pending, indicating scope for improving resolution efficiency
- A high number of unresolved tickets highlights the need for improved response and resolution processes
## Conclusion
This project highlights areas of improvement in customer support operations and helps in better decision-making.
